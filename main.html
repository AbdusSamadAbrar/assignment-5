Question-1: What are the different ways to select an element in the DOM?

DOM theke element select kora hoy getElementById() diye ID, getElementsByClassName() diye class, getElementsByTagName() diye tag name, ar querySelector() ba querySelectorAll() diye ID, class, o tag sobkichu select kora jay.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML tag shoho content dey, innerText visible text dey, ar textContent hidden text shoho puro text dey.

Question-3: What is event delegation in the DOM?

Event delegation hocche ekta parent element e event listener boshiye tar child elements gulor event handle kora.

Question-4: What is event bubbling in the DOM?

Event bubbling hocche ekta child element e event trigger hole se upore parent elements gulo sequentially propagate kora.

Question-5: How do you create, add, and remove elements using JavaScript?

Create: document.createElement('tag'), Add: parent.appendChild(child), Remove: element.remove().
